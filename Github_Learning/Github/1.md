# What is GitHub?

GitHub is a cloud platform for storing Git repositories.

# Create Repository

Example:
StudentManagementSystem

Click:
New Repository
Create repo.

# Connect Local Git to GitHub

Add remote:

git remote add origin https://github.com/username/repo.git

Verify:

git remote -v

Push code:

git push -u origin main

# Clone Repository

git clone https://github.com/username/repo.git
