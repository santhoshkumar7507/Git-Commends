# 1. What is Git?

Git is a distributed version control system used to track source code changes.

# 2. Difference between Git and GitHub?

Git = Version Control Tool

GitHub = Cloud Hosting Platform

# 3. What is Repository?

A project folder managed by Git.

# 4. What is Commit?

Snapshot of code at a specific point.

# 5. What is Branch?

Separate development line.

# 6. What is Merge?

Combining branches.

# 7. What is Pull Request?

Request to merge code changes.

# 8. What is Fork?

Copy of another repository.

# 9. Difference between Clone and Fork?

Clone:

Downloads repository

Fork:

Creates personal copy

# 10. What is HEAD?

Pointer to current commit.
